
За да се избере проект, десен бутон върху проекта -> Set as Startup Project

В отделните проекти има поне 2 два файла (с малки изключения), съдържащи функцията main,
поради което по подразбиране са изключени от VS проекта. За да стартирате отделните 
файлове, включете ги в проекта един по един, ноне заедно.
(десен бутон върху файла -> Include In Project / Exclude From Project)


Разясненията и условията към отделните програми са включени в съответния проект,
като файл с формат .pdf
